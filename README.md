# lte

## Log Codes

- 000 = Start Receiver
- 001 = Received UDP Telegram
- 002 = Exception

- 100 = Start Sender
- 101 = Send UDP Telegram

- 200 = Start GPS Logger
- 201 = GPS Coordinates
- 202 = GPS Speed in km/h

- 300 = Start Testrun
- 301 = Kill all processes
- 302 = Waiting for Process
- 303 = Pack logresults
- 304 = Kill one process

- 400 = Start LTE Modem Logger
